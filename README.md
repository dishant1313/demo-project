# TaskFlow — Modern Task Management Platform

A lightweight, AI-enhanced task management system built with React and Node.js for agile teams.

## Features
- 📋 Kanban board with drag-and-drop
- 👥 Team collaboration with real-time updates
- 📊 Sprint analytics and velocity tracking
- 🤖 AI-powered task estimation
- 🔔 Smart notifications and reminders

## Tech Stack
- **Frontend:** React 18, Tailwind CSS, Zustand
- **Backend:** Node.js, Express, Prisma ORM
- **Database:** PostgreSQL
- **Auth:** JWT + OAuth 2.0
- **AI:** Google Gemini API

## Getting Started
```bash
npm install
npm run dev
```
